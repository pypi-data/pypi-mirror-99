bds
***


Description
===========

CLI for the Big Data Service. Use this CLI to build, deploy, and
manage fully elastic Big Data Service build on Hadoop, Spark and Data
Science distribution, which can be fully integrated with existing
enterprise data in Oracle Database and Oracle Applications..
