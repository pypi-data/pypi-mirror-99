artifacts
*********


Description
===========

CLI covering the Registry services. Use this CLI to manage resources
such as container images and repositories.

==================

* container

  * configuration

    * get

    * update

  * image

    * delete

    * get

    * list

    * remove-version

    * restore

  * repository

    * change-compartment

    * create

    * delete

    * get

    * list

    * update
