Description
===========

Generates a manifest file for the objects that need to be exported


Usage
=====

   oci dts export generate-manifest [OPTIONS]


Required Parameters
===================

--bucket [text]

Name of the bucket for which export is requested

--compartment-id, -c [text]

Compartment OCID where the bucket resides

--job-id [text]

OCID of the Export Job


Optional Parameters
===================

--end [text]

The subset of objects that needs to be exported upto this object
(inclusive)

--prefix [text]

The subset of objects that needs to be exported whose names starts
with this Prefix

--start [text]

The subset of objects that needs to be exported starting with this
object (inclusive)
