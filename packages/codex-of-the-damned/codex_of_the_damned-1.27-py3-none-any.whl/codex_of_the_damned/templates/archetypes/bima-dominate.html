Highlights

This is an allies bleed deck using {{ bima }} with {{ dominate }} to play a profusion of {{ govern_the_unaligned }} and {{ deflection }}.

{{ bima }} gets no stealth but the sheer number of actions — and the fact that he has a strenght of 2 — makes it quickly difficult for the prey to block everything.

The deck has serious stealth, with most vampires having {{ obf }}, and even plays {{ cloak_the_gathering }} and {{ veil_the_legions }} that can beneficiate the {{ bima }}.

Most vampires of the crypt also have {{ DOM }} and build a nice chain for {{ govern_the_unaligned }}: {{ halim_bey }} can help get everyone out fast and cheap using the superior version.

Tips & Tricks

{{ waters_of_duat }} make for additional minions with {{ dom }} since the {{ dominate }} card is available in such numbers.

{{ conditioning }} is used for the lunge and can even be used by {{ bima }} if they didn't use {{ govern_the_unaligned }} beforehand.

The high number of {{ deflection }} and {{ delaying_tactics }} makes for a sturdy defence. Keeping a minion or two unlocked to play is critical.

Variants

The archetype is seldom played and not many variations exist. The number of {{ bima }} in respects to {{ waters_of_duat }} can vary and {{ perfectionist }} can be included to help the Followers of Set get some blood back. {{ ivor_blockleys_deck }} from 2008 is another example.
